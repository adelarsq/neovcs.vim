# neovcs.vim

VCS support for Neovim

Basic support for:

- [x] Git
- [x] Subversion

## Config

- `<leader>va` - add file to VCS
- `<leader>vA` - add file to VCS (custom)
- `<leader>vr` - remove file from VCS

- `<leader>vs` - VCS status

- `<leader>vb` - change branch

- `<leader>v?` - VCS help
